Welcome to the Bridgemate Data Connector help files

This pages explain how to use the Bridgemate ScoringProgramPipeClient to communicate with the Bridgemate Data Connector.

Getting Started

Important classses are:

  • The DataConnectorPipeClient class , used to connect to and disconnect from the Bridgemate Data Connector.

  • The ScoringProgramPipeClient class, used to handle communication with the Bridgemate Data Connector

  • The InitDTO class, used to relay initialization data for an event to the Bridgemate Data Connector

  • The ContinueDTO class , used to relay continuation data for an event to the Bridgemate Data Connector

  • The BCSManagementRequestDTO class , used to query information on the Bridgemate Control Software and to issue a "Shut Down Now" command to it.
